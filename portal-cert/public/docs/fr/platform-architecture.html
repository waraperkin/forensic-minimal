Architecture complète de la plateforme CYBERCORP Forensic Minimal : point d'entrée unique Nginx (TLS), portails CERT/IT, cluster OpenSearch, sidecars HELK et Velociraptor, stack CTI/IR et observabilité. Réseaux Docker : forensic-net, helk_net, velociraptor_net.

10. Sécurité & exploitation

  • TLS terminé sur Nginx — HSTS activé, certificats dans config/nginx/certs/
  • Session portail — cookie HttpOnly ; MFA TOTP optionnel
  • Portail IT — accès uniquement via token à durée/usage limités
  • Mode démo (Documentation) — bloque purge/delete/regenerate
  • Maintenance — ./forensic.sh check-health, docker compose logs cert-portal
