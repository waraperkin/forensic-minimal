La plateforme CYBERCORP Forensic v2.1 est un écosystème SOC/CERT intégré : portails analyste, ingestion forensique, recherche OpenSearch, investigation Timesketch, CTI (OpenCTI, MISP, TheHive, Cortex), détection (Sekoia.IO, SentinelOne) et observabilité (Grafana, Prometheus, Loki).

Périmètre fonctionnel

Portails

CERT (/) et IT (/it/) — upload, tokens, incidents, supervision.

Ingestion

Worker Python : parse EVTX/CSV/Plaso → OpenSearch + Timesketch + MinIO.

Recherche

Cluster OpenSearch 2 nœuds, Dashboards, Logstash, Filebeat.

Investigation

Timesketch (timelines), outils CERT (IOC, timeline builder, XDR view).

CTI

OpenCTI hub, MISP, TheHive cas, Cortex analyseurs.

Détection

Control planes Sekoia & SentinelOne via proxy /api/threat/*.

Démarrage & maintenance

  • ./forensic.sh start — démarrage rapide (FAST-BOOT)
  • ./forensic.sh full-start — activation complète (playbooks, dashboards, CTI)
  • ./forensic.sh urls — liste des accès et credentials de labo
  • ./forensic.sh fix-data — réparation OpenSearch / mappings
  • docker compose build cert-portal && docker compose up -d cert-portal — mise à jour portail
