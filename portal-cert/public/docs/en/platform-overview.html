The CYBERCORP Forensic v2.1 platform is an integrated SOC/CERT ecosystem: analyst portals, forensic ingestion, OpenSearch search, Timesketch investigation, CTI (OpenCTI, MISP, TheHive, Cortex), detection (Sekoia.IO, SentinelOne) and observability (Grafana, Prometheus, Loki).

Functional scope

Portals

CERT (/) and IT (/it/) — upload, tokens, incidents.

Ingestion

Python worker: EVTX/CSV/Plaso → OpenSearch + Timesketch + MinIO.

Search

OpenSearch 2-node cluster, Dashboards, Logstash, Filebeat.

Investigation

Timesketch timelines, CERT tools (IOC, timeline builder, XDR view).

CTI

OpenCTI hub, MISP, TheHive cases, Cortex analyzers.

Detection

Sekoia & SentinelOne control planes via /api/threat/*.
